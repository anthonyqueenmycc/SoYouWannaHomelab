# Creating Your First Virtual Machine
Now that you have VirtualBox installed, the next step is to create a virtual machine! Before we get started with that though, we need to get an operating system to run on it. My recommendation is start with Linux Mint. Like the name implies, it is a Linux operating system, but it also does not require a large amount of resources from your host computer. So we will walk through the steps of downloading the ISO file, and creating a virtual machine to run it on.<br>
<br>
1. Let's start by downloading Linux Mint. Click <a href="https://linuxmint.com/">HERE</a> to go to their website, the click the white "Download" button.<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/7df3362d-7fa1-4b34-b6f4-857511f6bcb4" /><br>
<br>
2. On the downloads page, click the black "Download" button for the Cinnamon Edition of Linux Mint.<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/49410efd-3ab9-4c1f-984f-e397f1879087" /><br>
<br>
3. On the next page, you will need to scroll down and choose a location to download the disc image from. Choose a location in the same country as you for the quickest download.<br>
<br>
<img width="1842" height="2081" alt="mirror" src="https://github.com/user-attachments/assets/6534d5b0-15ac-4d16-887f-e474653ff6e7" /><br>
<br>
4. Wait for the download to finish. Now would be a good time to work on assignments for class!<br>
<br>
5. Once the download has finished, minimize the web browser, and open up VirtualBox. There should be a shortcut on your desktop if you closed out of it after the installation finished previously.<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/6bc61b23-9faa-4adf-8f82-12b510a1891e" /><br>
<br>
6. With VirtualBox now open, click the the blue button labeled "New" to launch the virtual machine creation wizard.<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/351730cc-c234-4344-b028-43f4af5e79b9" />
<br>
7. In the New Virtual Machine wizard window, start by giving your new VM a name. A good name should let us be able to easily identify the device without launching it. I will call mine "Linux Mint".<br>
<br>
<img width="796" height="570" alt="image" src="https://github.com/user-attachments/assets/37296f1d-253a-4a42-b241-e4b49166ec03" /><br>
<br>
8. Next, we need to select the Linux Mint file we downloaded previously. In the section titled "ISO Image", click the dropdown arrow, and then click "Other".<br>
<br>
<img width="798" height="571" alt="image" src="https://github.com/user-attachments/assets/ae983cb6-86c3-4f5d-91ee-92a9a0158745" /><br>
<br>
9. Navigate to your downloads folder, select the Linux Mint file, and then click "Open".<br>
<br>
<img width="634" height="481" alt="image" src="https://github.com/user-attachments/assets/1f597ce2-2d89-453e-b5a6-62dc6e19d100" /><br>
<br>
10. Once we are back on the New Virtual Machine wizard, we want to <b>UNCHECK</b> the Unattended Installation box, then click on the "Specify virtual hardware" section.<br>
<br>
<img width="795" height="566" alt="image" src="https://github.com/user-attachments/assets/17bfbc87-d7dc-4934-8465-5126f3bc71aa" /><br>
<br>
11. For the memory, we will allocate 2048 MB, and leave the number of CPUs at 1. This is the minimum required to run Linux Mint. It won't be the fastest, but it will work for what we need it to do. Once you have those values set, click the section for "Specify virtual hard disk"<br>
<br>
<img width="796" height="568" alt="image" src="https://github.com/user-attachments/assets/087cc1b7-1870-4dfd-9542-03d5d6c30211" /><br>
<br>
12. We can leave the disk size at the default of 25.00 GB. All other settings here should stay the same as well. Go ahead and click "Finish".<br>
<br>
<img width="794" height="567" alt="image" src="https://github.com/user-attachments/assets/b02139cf-d9ce-4956-8974-96ec9f93b576" /><br>
<br>
13. Your virtual machine should now be showing in VirtualBox! Go ahead and double-click it to launch it.<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/8e8b6b95-fc4d-492b-959a-44152c2ddc10" /><br>
<br>
14. Once the virtual machine launches, you click the red x in the corner to get rid of the slide out notification menu, then press Enter to select the option "Start Linux Mint". You can also click the square icon at the top of the window to make your machine full-screen.<br>
<br>
<img width="649" height="572" alt="image" src="https://github.com/user-attachments/assets/eb2138dc-cbd8-4ac1-b502-8db9dd43881c" /><br>
<br>
15. Wait several moments while the operating system loads. The screen may go black for a few minutes, this is normal when we are using the minimum amount of RAM and CPUs.<br>
<br>
16. You will eventually see the Linux Mint desktop. Double click on the icon that says "Install Linux Mint".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/a051d9b2-e984-408e-9e12-3a4bd54c1677" /><br>
<br>
17. The installation wizard will launch and help guide us through the installation. Start by choosing your language, and then clicking "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/ee70140e-d854-4f8b-83c9-bda1decc0234" /><br>
<br>
18. Next we need to pick a keyboard layout. The default selections should be fine, so go ahead and click "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/b16455fc-7b22-4464-a16f-02fb279ef1ed" /><br>
<br>
19. The next page will give you an option for installing multimedia codecs. Since we won't be watching videos or listening to music on this virtual machine, we can leave everything as it is and just click "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/49d19593-bc5b-405a-8c2f-22b6a6518c44" /><br>
<br>
20. We are now presented with the Installation type page. Leave everything as it is, and click "Install Now".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/bc635c86-4b1d-4e0b-87ff-9229e5dfd9ca" /><br>
<br>
21. You will receive a pop-up asking you to confirm the changes to the disk. Click "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/23257570-d6d4-4535-a1cd-bb5045c3f685" /><br>
<br>
22. The wizard will now ask for your location so it can automatically choose the correct timezone. Click on the map, or type in the name of your nearest major city, and then click "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/d6e0893f-374f-4781-98cf-67b699784187" /><br>
<br>
23. You will now be given the chance to enter in several pieces of information.<br>
"Your name" is how your name is displayed in the operating system.<br>
"Your computer's name" is how other devices on the network will see this virtual machine.<br>
"Pick a username" is the username you will use to log in with.<br>
"Choose a password/Confirm your password" is where you set the password for your account. Since this is just a test device, I recommend making it "Passw0rd". If you forget the password, <b> THERE IS NO WAY TO RECOVER IT!</B> You will just have to recreate the machine from scratch.<br>
Once you have filled all those fields out, go ahead and click "Continue".<br>
<br>
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/f9878f03-6475-4f9f-8c39-444a49cc5ed0" /><br>
<br>
24. Now you play the waiting game! It may take a while for the operating system to be installed to the virtual machine. Take this opportunity to work on some assignments!
